# SuiSnap - Trilha Didática

Estrutura por módulos para ensinar Move e Sui usando o encurtador de links SuiSnap. Cada módulo tem exercícios práticos.

- `modulo-0-setup`: ambiente, Sui CLI, carteira e primeiro pacote.
- `modulo-1-programmable-object-model`: objeto `Link`, UID, mutabilidade e transferências.
- `modulo-2-ownership-autorizacao`: `tx_context::sender`, capabilities, restrições por endereço.
- `modulo-3-modelagem-move`: módulos, imports, vetores para strings, boas práticas.
- `modulo-4-eventos`: eventos `LinkCreated/Updated/Deleted`, observabilidade.
- `modulo-5-testes`: `sui move test`, casos de sucesso e falha, TDD.
- `modulo-6-frontend`: Sui dApp Kit, moveCall, listar/renderizar objetos, UX.
- `modulo-7-extensoes`: dynamic fields, EditCap granular, NFT, hash, expiração, pagamentos.
